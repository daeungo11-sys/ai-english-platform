## 🧩 컴포넌트 구조

### 1. App.tsx (루트 컴포넌트)
- **역할**: 라우팅 및 네비게이션 관리
- **기능**:
  - React Router로 페이지 라우팅
  - 상단 네비게이션 바 렌더링
  - 활성 페이지 하이라이트

### 2. Home.tsx
- **역할**: 플랫폼 소개 및 기능 안내
- **기능**:
  - 말하기/쓰기/읽기 기능 소개 카드
  - 각 기능별 링크 제공
  - 통계 정보 표시

### 3. Speaking.tsx
- **역할**: 영어 말하기 연습
- **주요 기능**:
  - 마이크 녹음 (MediaRecorder API)
  - 3분 타이머
  - 오디오 재생
  - AI 피드백 (발음, 문법, 종합 평가)

### 4. Writing.tsx
- **역할**: 영어 쓰기 연습
- **주요 기능**:
  - 텍스트 에디터
  - 10분 타이머
  - 단어 수 카운터
  - AI 피드백 (문법, 어휘, 구조, 종합 평가)

### 5. Reading.tsx
- **역할**: 영어 읽기 연습
- **주요 기능**:
  - TOEIC/TOEFL/수능 모드 선택
  - 지문 표시
  - 객관식 문제
  - 문제당 2분 타이머
  - AI 채점 및 해설

### 6. Calendar.tsx
- **역할**: 학습 기록 관리
- **주요 기능**:
  - 동적 달력 생성
  - 날짜별 학습 기록 추가/수정/삭제
  - 영역별 색상 표시
  - 난이도 평가
  - 메모 작성

## 🔄 데이터 흐름

```
사용자 입력
    │
    ├─→ Speaking: 녹음 파일 → Blob → AI 피드백
    │
    ├─→ Writing: 텍스트 → AI 피드백
    │
    ├─→ Reading: 선택 답안 → 채점 → 해설
    │
    └─→ Calendar: 기록 데이터 → 로컬 State → UI 업데이트
```

## 🎨 스타일 구조

```
index.css (글로벌)
    │
    ├─→ CSS 변수 정의 (색상, 간격 등)
    ├─→ 전역 스타일
    └─→ 공통 컴포넌트 스타일 (버튼, 카드 등)

App.css
    │
    └─→ 네비게이션 바 스타일
        └─→ 레이아웃 스타일

pages/*.css
    │
    ├─→ 각 페이지별 고유 스타일
    └─→ 반응형 디자인
```

## 🔌 주요 기술 스택

### Frontend
- **React 18**: UI 라이브러리
- **TypeScript**: 타입 안정성
- **React Router**: 클라이언트 사이드 라우팅
- **Vite**: 빌드 도구 및 개발 서버
- **Lucide React**: 아이콘 라이브러리

### 기능별 API 사용
- **MediaRecorder API**: 오디오 녹음 (Speaking 페이지)
- **Date API**: 달력 생성 및 날짜 관리 (Calendar 페이지)

## 📊 상태 관리

### 로컬 State (useState)
- 각 페이지 컴포넌트에서 자체 상태 관리
- 타이머, 입력값, 피드백 등

### 데이터 저장
- 현재: 브라우저 메모리 (페이지 새로고침 시 초기화)
- 향후: localStorage 또는 백엔드 연동 가능
