# AI 영어 교육 플랫폼 - 기능 중심 구조 가이드

이 문서는 프로젝트를 기능 단위로 설명하여 새로운 개발자가 빠르게 이해할 수 있도록 작성되었습니다.

## 🎯 플랫폼 개요

AI 영어 교육 플랫폼은 **말하기**, **쓰기**, **읽기** 세 가지 영어 학습 기능과 **학습 기록 관리** 기능을 제공하는 웹 애플리케이션입니다.

---

## 📱 주요 기능 및 구현

### 1. 🎤 말하기 연습 기능

#### 기능 설명
사용자가 일일 질문에 대해 3분 동안 답변을 녹음하고, AI 튜터가 발음과 문법을 교정해주는 기능입니다.

#### 사용자 플로우
```
1. 말하기 페이지 접속
   ↓
2. 오늘의 질문 확인
   ↓
3. "녹음 시작" 버튼 클릭
   ↓
4. 3분 동안 답변 녹음
   ↓
5. "녹음 중지" 버튼 클릭
   ↓
6. 녹음본 재생 확인
   ↓
7. "피드백 받기" 버튼 클릭
   ↓
8. AI 피드백 확인 (발음, 문법, 종합 평가)
```

#### 구현 위치
- **페이지 컴포넌트**: `src/pages/Speaking.tsx`
- **스타일**: `src/pages/Speaking.css`
- **주요 기능 구현**:
  - 녹음 기능: MediaRecorder API 사용
  - 타이머: useState + useEffect로 구현
  - 오디오 재생: Blob URL 생성
  - 피드백: 현재는 시뮬레이션 데이터 (향후 AI API 연동 예정)

#### 핵심 상태 관리
```typescript
- timeLeft: 남은 시간 (초 단위, 초기값 180)
- isRecording: 녹음 중 여부
- audioBlob: 녹음된 오디오 데이터
- feedback: AI 피드백 결과
```

#### 주요 기능 상세
- **타이머**: 녹음 시작 시 자동 시작, 0초가 되면 자동 중지
- **녹음**: 브라우저 마이크 접근 권한 필요
- **피드백**: 발음, 문법, 종합 평가 세 가지 카테고리

### 2. ✍️ 쓰기 연습 기능

#### 기능 설명
사용자가 주어진 주제에 대해 10분 동안 에세이를 작성하고, AI가 문법, 어휘, 구조, 종합 평가를 제공하는 기능입니다.

#### 사용자 플로우
```
1. 쓰기 페이지 접속
   ↓
2. 오늘의 주제 확인
   ↓
3. "작성 시작" 버튼 클릭
   ↓
4. 10분 동안 에세이 작성 (단어 수 실시간 표시)
   ↓
5. "피드백 받기" 버튼 클릭
   ↓
6. AI 피드백 확인 (문법, 어휘, 구조, 종합 평가)
```

#### 구현 위치
- **페이지 컴포넌트**: `src/pages/Writing.tsx`
- **스타일**: `src/pages/Writing.css`
- **주요 기능 구현**:
  - 텍스트 에디터: textarea 요소
  - 단어 수 계산: 공백 기준 실시간 계산
  - 타이머: 10분 제한 (600초)
  - 피드백: 4가지 카테고리 (문법, 어휘, 구조, 종합)

#### 핵심 상태 관리
```typescript
- timeLeft: 남은 시간 (초 단위, 초기값 600)
- text: 작성 중인 텍스트
- wordCount: 단어 수 (실시간 계산)
- isActive: 작성 모드 활성화 여부
- feedback: AI 피드백 결과
```

#### 주요 기능 상세
- **단어 수 카운터**: 텍스트 변경 시 실시간 업데이트
- **타이머**: 시간 초과 시 입력 비활성화
- **피드백**: 문법, 어휘, 구조, 종합 평가 제공

### 3. 📖 읽기 연습 기능

#### 기능 설명
사용자가 TOEIC, TOEFL, 수능 지문을 읽고 문제를 풀며, 각 문제의 해결 방법을 학습하는 기능입니다.

#### 사용자 플로우
```
1. 읽기 페이지 접속
   ↓
2. 시험 유형 선택 (TOEIC/TOEFL/수능)
   ↓
3. 지문 읽기
   ↓
4. 문제당 2분 안에 답안 선택
   ↓
5. 즉시 채점 및 해설 확인
   ↓
6. 다음 문제로 이동
```

#### 구현 위치
- **페이지 컴포넌트**: `src/pages/Reading.tsx`
- **스타일**: `src/pages/Reading.css`
- **주요 기능 구현**:
  - 시험 유형 선택: 버튼으로 전환
  - 지문 표시: 스크롤 가능한 영역
  - 문제 풀이: 객관식 선택
  - 타이머: 문제당 2분 (120초)
  - 채점 및 해설: 즉시 피드백

#### 핵심 상태 관리
```typescript
- currentQuestion: 현재 문제 번호
- timeLeft: 남은 시간 (초 단위, 초기값 120)
- selectedAnswer: 선택한 답안
- answers: 제출한 답안 목록
- showFeedback: 피드백 표시 여부
- examType: 선택한 시험 유형
```

#### 주요 기능 상세
- **시험 유형**: TOEIC, TOEFL, 수능 세 가지
- **타이머**: 문제당 2분, 시간 초과 시 자동 제출
- **채점**: 정답/오답 즉시 표시
- **해설**: 각 문제에 대한 상세 설명 제공

### 4. 📅 학습 기록 관리 기능

#### 기능 설명
사용자가 매일의 학습 경험을 달력에 기록하고, 어려웠던 부분을 추적할 수 있는 기능입니다.

#### 사용자 플로우
```
1. 학습 기록 페이지 접속
   ↓
2. 달력에서 날짜 클릭
   ↓
3. 모달에서 기록 입력
   - 영역 선택 (말하기/쓰기/읽기)
   - 난이도 선택 (하/중/상)
   - 메모 작성
   ↓
4. 저장 버튼 클릭
   ↓
5. 달력에 색상 표시 (영역별 다른 색상)
   ↓
6. 기록 목록에서 확인 및 수정/삭제 가능
```

#### 구현 위치
- **페이지 컴포넌트**: `src/pages/Calendar.tsx`
- **스타일**: `src/pages/Calendar.css`
- **주요 기능 구현**:
  - 동적 달력 생성: 현재 월 자동 계산
  - 기록 추가/수정/삭제: 모달 기반
  - 영역별 색상: 말하기(파란색), 쓰기(주황색), 읽기(녹색)
  - 난이도 표시: 하(녹색), 중(주황색), 상(빨간색)

#### 핵심 상태 관리
```typescript
- entries: 학습 기록 배열
- showModal: 모달 표시 여부
- selectedDate: 선택한 날짜
- selectedEntry: 수정 중인 기록
- formData: 폼 입력 데이터
```

#### 데이터 구조
```typescript
interface DiaryEntry {
  id: string              // 고유 ID
  date: string            // 날짜 (YYYY-MM-DD)
  type: 'speaking' | 'writing' | 'reading'  // 학습 영역
  difficulty: '하' | '중' | '상'  // 난이도
  notes: string           // 메모
}
```

#### 주요 기능 상세
- **달력 생성**: 현재 월을 기준으로 동적 생성 (월의 첫 번째 날의 요일 계산)
- **기록 표시**: 기록이 있는 날짜는 색상 점으로 표시
- **CRUD 기능**: 생성, 읽기, 수정, 삭제 모두 지원
- **영역별 색상**: 각 학습 영역마다 다른 색상으로 구분

## 🔄 기능 간 상호작용

### 공통 기능

#### 1. 네비게이션
- **위치**: `src/App.tsx`
- **기능**: 모든 페이지에서 상단 네비게이션 바 제공
- **특징**: 
  - 활성 페이지 하이라이트
  - 글래스모피즘 디자인
  - 반응형 메뉴

#### 2. 응원 문구
- **위치**: 각 페이지 컴포넌트
- **기능**: 사용자 동기부여를 위한 메시지 표시
- **특징**:
  - 말하기: "매일 영어로 말하는 연습을 하면 자신감이 쑥쑥 자라납니다!"
  - 쓰기: "글을 쓰는 것은 생각을 정리하는 가장 좋은 방법입니다!"
  - 읽기: "꾸준히 읽으면 어휘력과 이해력이 자연스럽게 향상됩니다!"
  - 학습 기록: "매일의 작은 기록이 큰 성장을 만들어냅니다!"

#### 3. 타이머 기능
- **공통 패턴**: 모든 연습 기능에서 사용
- **구현 방식**:
  ```typescript
  useEffect(() => {
    const interval = setInterval(() => {
      setTimeLeft(prev => prev - 1)
    }, 1000)
    return () => clearInterval(interval)
  }, [timeLeft])
  ```
- **차이점**:
  - 말하기: 3분 (180초)
  - 쓰기: 10분 (600초)
  - 읽기: 문제당 2분 (120초)
